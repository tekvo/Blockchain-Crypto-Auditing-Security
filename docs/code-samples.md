# Code samples

The Solidity under **`codes/`** at the **repository root** (next to `docs/` and `mkdocs.yml`) includes ERC20-related examples, access control, pausable patterns, and utilities such as `SafeMath` and `EnumerableSet`.

Treat everything there as **educational snippets**, not production deployments.

## Layout

- **`codes/Examples/`** — Example contracts (ERC20 presets, access control demos).
- **`codes/interfaces/`** — Interface and security-related contract interfaces (ERC20, access control, pausable).
- **`codes/utils/`** — Helpers (`Context`, `SafeMath`, `String`, `EnumerableSet`).

Clone or download the repository and open these paths in your editor. The static documentation site does not embed that tree; it stays versioned alongside the docs.
